## 📖 Story Synopsis

> *"You can go back as many times as you want... but there's no such thing as a happy ending without sacrifice."*

```
    🏠 Modern Japan          🌀 Portal          🏰 Fantasy World
         |                      |                     |
         |                      |                     |
    Subaru Natsuki ────────────┼─────────────> New Reality
         |                      |                     |
         |                      |                     ↓
         |                      |              💀 Death
         |                      |                     |
         |                      |                     ↓
         └──────────────────────┴─────────────── 🔄 Return by Death
```

### The Premise

**Subaru Natsuki**, an ordinary NEET, is suddenly transported to a fantasy world. Unlike typical isekai heroes, he receives no overpowered abilities—except one mysterious power: **Return by Death**.

```
╔═══════════════════════════════════════════════════════════════╗
║  ⚡ RETURN BY DEATH                                           ║
╠═══════════════════════════════════════════════════════════════╣
║                                                               ║
║  ▸ Subaru dies ────────────────────────────┐                 ║
║                                            │                 ║
║  ▸ Time rewinds to checkpoint ◄────────────┘                 ║
║                                                               ║
║  ▸ Only Subaru retains memories                              ║
║                                                               ║

║    (Penalty: Death or worse)                                 ║

│        ┃                                 ┃                              │
│        ┃                                 ┃                              │
│   ┌────▼────────┬────────────────┬───────▼────────┬──────────────┐    │
│   │  Timeline 1 │  Timeline 2    │  Timeline 3    │ Timeline N   │    │
│   └─────────────┴────────────────┴────────────────┴──────────────┘    │
│        │             │                  │               │              │
│        ├─────────────┼──────────────────┼───────────────┤              │
│        │             │                  │               │              │
│        ▼             ▼                  ▼               ▼              │
│    ATTEMPT 1     ATTEMPT 2          ATTEMPT 3      SUCCESS???         │
│     (Failed)     (Failed)           (Failed)       (Maybe...)         │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

        Each death = Knowledge gained + Mental trauma accumulated
```
